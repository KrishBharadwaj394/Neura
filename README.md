**Neura** is a powerful AI chatbot that provides prompt-based answers, text-to-speech functionality, and the ability to download responses as PDF documents. It also features a dynamic theme toggle for light and dark modes, offering a personalized user experience.

## Features

- **AI-Powered Answers**: Get prompt-based responses generated by a powerful AI model.
- **Text-to-Speech**: Convert generated responses into speech, making it interactive and engaging.
- **PDF Export**: Download the AI-generated responses in PDF format for offline use.
- **Dark & Light Mode**: Switch between dark and light themes to suit your preference.
